Habilidades

Python

Python é uma linguagem de programação poderosa e fácil de aprender. Ela é amplamente utilizada para desenvolvimento web, automação, ciência de dados, e muito mais.

C#

C# é amplamente usada para desenvolvimento de aplicações de desktop, web e mobile, além de ser uma escolha comum para jogos com o motor Unity.

SQL

SQL permite criar, ler, atualizar e deletar dados em bancos de dados, sendo uma habilidade essencial para desenvolvedores e analistas de dados.

Robotica

A robótica é a ciência que envolve a criação de máquinas que podem executar tarefas de forma autônoma. Com aplicações em setores como indústria, saúde e entretenimento, a robótica está revolucionando a maneira como interagimos com a tecnologia.

MMC
(Montagem e manutenção de computadores)

MMC (Montagem e Manutenção de Computadores) é uma área que se concentra na construção e manutenção de sistemas de computação. Profissionais dessa área montam, configuram e realizam reparos em computadores, garantindo que eles funcionem de maneira eficiente e estável.

Front-End

O front-end é a parte de um site ou aplicativo com a qual o usuário interage diretamente. Envolve o design e a programação de interfaces utilizando HTML, CSS e JavaScript, garantindo uma experiência visual atraente e intuitiva.
